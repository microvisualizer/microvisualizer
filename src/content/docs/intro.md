MicroVisualizer is a tool for visualizing microservices architecture. Define services, messaging topics, and their relationships to generate an interactive graph of your system.

## Getting Started

- [API Reference](/api/) — Browse the full OpenAPI specification.

## OpenAPI specification

The spec is hosted at a stable URL — plug it into Postman, openapi-generator, or any OpenAPI tool:

```
https://docs.microvisualizer.com/microvisualizer-v1.yaml
```

Or [download it](/microvisualizer-v1.yaml) directly.
